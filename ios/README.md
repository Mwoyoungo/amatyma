# Amatyma — iOS (SwiftUI)

The native iOS app. **Real app**, sharing the **same Firebase backend** as the Android
app (Firestore / Storage / Functions / FastPix / push). See `../IOS_HANDOFF.md` for the
full social/chat separation map and the CometChat bridge.

The Xcode project is generated by **XcodeGen** from `project.yml` (so the repo stays clean
and it builds the same on any Mac or CI — no committed `.xcodeproj`).

## Build it (on a Mac or macOS CI — no Apple Developer account needed for the Simulator)

```bash
brew install xcodegen          # one time
cd ios
xcodegen generate              # creates Amatyma.xcodeproj
open Amatyma.xcodeproj          # to run in Xcode, OR build headless:

xcodebuild -scheme Amatyma -sdk iphonesimulator \
  -destination 'platform=iOS Simulator,name=iPhone 15' build
```

Running in the **iOS Simulator requires no signing and no Apple Developer Program**.
The $99/yr program is only needed later for TestFlight, real devices, and the App Store.

## Firebase config (required to run)

1. Firebase console → add an **iOS app** with bundle id **`com.lokaleza.amatyma`**.
2. Download **`GoogleService-Info.plist`** → place it at `ios/Amatyma/GoogleService-Info.plist`
   (it is gitignored — add per machine).

## Structure
- `Amatyma/AmatymaApp.swift` — entry point, Firebase configure
- `Amatyma/Models/` — data models (mirror the Android `social` models)
- `Amatyma/Stores/` — `ObservableObject` stores with Firestore listeners (mirror Likes/Following/Feed)
- `Amatyma/Views/` — SwiftUI screens (Feed, Auth, …)
- `Amatyma/Support/` — theme & helpers
